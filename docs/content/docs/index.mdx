## Welcome to RADSAT-SK2 Software documentation!

**RADSAT-SK2** (Radiation Satellite Saskatchewan 2) is the second cube satellite designed at the University of Saskatchewan.

Find more about the project at our [main website](https://skcubesat.ca/).

Check out our organization GitHub page and other repositories at [SkCubeSat](https://github.com/SkCubeSat).

To get started, check out the [Getting Started](./setup/getting-started) section for further information, including how to set up the project and contribute.

> This project is under active development.
