---
title: "Kubos Libraries"
---

Kubos provides the following libraries to assist with core functionality:

- [Applications API](../../ecosystem/apps/app-guide) - Simplifies the process of setting up a mission application

  > - [Python](../../ecosystem/apps/python-app-api)
  > - <a href="../../rust-docs/kubos_app/index.html" target="_blank">Rust</a>

- [Kubos Service Library](../../ecosystem/services/service-dev) - Provides a base framework for KubOS services

  > - [Python](https://github.com/kubos/kubos/tree/master/libs/kubos-service)
  > - <a href="../../rust-docs/kubos_service/index.html" target="_blank">Rust</a>

- <a href="../../rust-docs/radsat_system/index.html" target="_blank">Kubos System</a> - Provides helper functions for things like reading configuration files and setting U-Boot environment variables

- <a href="../../rust-docs/comms_service/index.html" target="_blank">Communications Service Framework</a> - Provides a base framework to assist with implementing the system's communications service

- <a href="../../rust-docs/file_protocol/index.html" target="_blank">File Protocol</a> - Used by the file service in order to process simultaneous, asynchronous file transfer requests

- <a href="../../rust-docs/shell_protocol/index.html" target="_blank">Shell Protocol</a> - Used by the shell service in order to handle simultaneous, asynchronous remote shell connections

- <a href="../../rust-docs/cbor_protocol/index.html" target="_blank">CBOR Protocol</a> - Constructs and processes the CBOR packets which are used for sending data to and from the file and shell services

- <a href="../../rust-docs/channel_protocol/index.html" target="_blank">Channel Protocol</a> - Sends and receives CBOR packets for the file and shell services, maintaining ownership and separation between simultaneous client connections

>
